# Game-Store
A professional e-commerce platform built to manage a digital game storefront. This project features a responsive UI, secure role-based authentication (Admin/Customer), and a dynamic shopping cart integrated with a backend database for real-time inventory and order tracking.

**Project Overview**
This application demonstrates a complete software development lifecycle, from UI design to backend security and database integration.

**Responsive Design**: Interactive forms for game browsing, product details, and cart management that adapt to any screen size.

**Role-Based Access Control (RBAC)**: Secure login/signup system using password hashing; Admins manage inventory while Customers handle purchases.

**Core E-commerce Logic**: Dynamic cart calculations, simulated payment processing, and automated stock updates upon checkout.

**Database Management**: Utilizes stored procedures for secure credential validation and order history persistence.
